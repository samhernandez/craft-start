# Simple Craft CMS Starter

My own simple boilerplate with

- Craft CMS
- Vite - vanilla js
- Tailwind CSS

## Steps

1. Clone this repository
1. Delete the `.git` directory
1. Find/replace `you@email.com` in `/config/project` with your email address
1. Run `npm install`
1. Install Craft

## Front-end dev

Vite runs on port `3000`

```
npm run dev
npm run build
```
